# AWay App - Store Submission Guide

## App Store (iOS) Requirements

### App Information
- **App Name**: AWay - Motorcycle Community
- **Subtitle**: Connect with riders worldwide
- **Category**: Social Networking (Primary), Travel (Secondary)
- **Content Rating**: 12+ (Infrequent/Mild Mature/Suggestive Themes)

### Description (Italian)
```
AWay è l'app definitiva per i motociclisti! Connettiti con rider di tutto il mondo, scopri eventi, pianifica viaggi e condividi la tua passione per le due ruote.

CARATTERISTICHE PRINCIPALI:

🏍️ FEED SOCIAL
Condividi foto, storie e momenti della tua vita da motociclista. Segui altri rider e scopri contenuti ispiranti dalla community.

📅 EVENTI
Scopri raduni, tour organizzati e meetup nella tua zona. Crea i tuoi eventi e invita altri rider.

🗺️ TRIP PLANNER AI
Pianifica il tuo prossimo viaggio con l'intelligenza artificiale. Ottieni itinerari personalizzati basati sulle tue preferenze.

🛒 MERCATO
Compra e vendi moto, accessori e abbigliamento. Trova l'affare perfetto o metti in vendita i tuoi articoli.

🏁 TRACCE GPX
Scarica e condividi percorsi GPS. Esplora le strade più belle del mondo.

🚨 SOS & SICUREZZA
Funzione di emergenza con rilevamento incidenti e notifica automatica ai contatti fidati.

🏠 GARAGE DIGITALE
Gestisci la tua collezione di moto. Tieni traccia di manutenzioni e chilometri.

🌍 COMMUNITY GLOBALE
Connettiti con rider da tutto il mondo. Filtra per paese e tipo di moto.

Scarica AWay oggi e unisciti alla più grande community di motociclisti!
```

### Keywords
```
moto, motociclisti, rider, community, social, eventi, viaggi, gpx, tracce, sos, sicurezza, garage, mercato, trip planner
```

### Screenshots Required
1. **Feed Screen** - 6.5" and 5.5" iPhone screenshots
2. **Events Screen** - Shows event listing
3. **Trip Planner** - AI trip planning feature
4. **Market** - Buy/sell listings
5. **Profile** - User profile with garage
6. **SOS Feature** - Safety feature showcase

## Pre-Submission Checklist

### Technical
- [ ] App tested on multiple devices
- [ ] No crashes or major bugs
- [ ] All permissions properly declared
- [ ] Deep links working (if applicable)
- [ ] Push notifications configured (if applicable)

### Content
- [ ] All placeholder text replaced
- [ ] No test data in screenshots
- [ ] Professional screenshots created
- [ ] App description proofread
- [ ] Keywords optimized

### Legal
- [ ] Privacy Policy published
- [ ] Terms of Service published
- [ ] GDPR compliance (delete account feature)
- [ ] Age rating questionnaire completed

### Assets
- [ ] App icon in all required sizes
- [ ] Feature graphic (Play Store)
- [ ] Screenshots for all device sizes
- [ ] Demo video (optional but recommended)

## EAS Build Commands

### Development Build
```bash
eas build --profile development --platform all
```

### Preview Build (Internal Testing)
```bash
eas build --profile preview --platform all
```

### Production Build
```bash
eas build --profile production --platform all
```

### Submit to Stores
```bash
# iOS App Store
eas submit --platform ios

# Google Play Store
eas submit --platform android
```
